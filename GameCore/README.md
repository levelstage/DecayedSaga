# GameCore
순수 C# 로직 라이브러리. MonoGame 의존성 없음.

| 디렉토리 | 역할 |
|----------|------|
| `Battle/` | 전투 시스템 전체 (그리드, 페이즈, 피해, 제압, 트리거) |
| `Behaviors/` | 씬 무관 범용 실행 파이프라인 (BehaviorQueue) |
| `Events/` | 발행-구독 이벤트 버스 |
| `Map/` | 던전 노드 그래프 |
| `Scripting/` | 스킬 인터페이스 + 리플렉션 바인딩 |
| `Units/` | 유닛·장비·카드 데이터 모델 |
